O que é DNS flush e por que você deveria fazer

InsNer HUB  ·  março de 2026  ·  6 min de leitura

Se você está enfrentando lentidão na internet, sites que não carregam ou problemas de conexão em jogos online, o DNS flush pode ser exatamente o que você precisa. Neste artigo, explicamos o que é o cache DNS, por que ele pode causar problemas e como limpá-lo no Windows.

O QUE É O DNS?

O DNS (Domain Name System) funciona como uma agenda telefônica da internet. Quando você digita um endereço como "google.com", seu computador precisa saber qual é o endereço IP real desse servidor para se conectar a ele. O DNS é o serviço responsável por fazer essa tradução.

Para evitar fazer essa consulta toda vez que você acessa um site, o Windows armazena as respostas em um cache DNS local — uma espécie de memória temporária com os endereços já consultados.

O QUE É DNS FLUSH?

O DNS flush (ou limpeza de cache DNS) é o processo de apagar todas as entradas armazenadas no cache DNS do seu computador. Com o cache limpo, o Windows volta a consultar os servidores DNS para obter os endereços atualizados.

QUANDO O CACHE DNS CAUSA PROBLEMAS?

O cache DNS pode ficar desatualizado ou corrompido em diversas situações:

  • Um site mudou de servidor e o IP antigo ficou salvo no cache
  • Entradas corrompidas por instabilidade de conexão
  • Malware que modificou entradas do DNS para redirecionar tráfego
  • Mudança de provedor de internet ou configurações de rede
  • Alta latência em jogos online por endereços de servidor desatualizados

💡 Se um site funciona no celular mas não no PC, ou se você vê erros como "DNS_PROBE_FINISHED_NXDOMAIN", o DNS flush costuma resolver.

COMO FAZER DNS FLUSH NO WINDOWS

O processo é simples e leva menos de um minuto. Abra o Prompt de Comando como administrador (clique com o botão direito no menu Iniciar → Terminal do Windows como administrador) e execute os seguintes comandos:

ipconfig /flushdns

Esse comando limpa o cache DNS. Você verá a mensagem "Cache do DNS Resolver liberado com êxito."

ipconfig /registerdns

Registra novamente todos os nomes DNS, forçando uma atualização completa.

ipconfig /release
ipconfig /renew

Libera e renova o endereço IP da sua conexão — útil quando o problema é mais amplo que apenas o DNS.

O QUE É WINSOCK RESET?

O Winsock é uma interface do Windows que gerencia as conexões de rede dos programas. Com o tempo, ele pode ficar corrompido por instalações de software, malware ou falhas do sistema, causando problemas de conectividade mesmo com a internet funcionando normalmente.

O reset do Winsock restaura essas configurações para o padrão. Para fazer manualmente:

netsh winsock reset
netsh int ip reset

Após executar, reinicie o computador para que as mudanças entrem em vigor.

COM QUE FREQUÊNCIA FAZER?

Não existe uma frequência obrigatória — faça quando notar problemas de conexão. Muitos usuários fazem mensalmente como parte de uma rotina de manutenção. Para jogadores, fazer antes de sessões importantes pode reduzir a latência inicial.
